# Controle de Qualidade

## Análise Estática
- Avaliação de documentação e código-fonte
- Métodos formais

## Análise Dinâmica
- Técnicas aplicadas com o código em execução

## Verificação x Validação

### Verificação
Garantir que o produto está sendo construído corretamente.
- Code review
- Ferramentas de análise de código
- Revisão de histórias

### Validação
Garantir que o produto correto está sendo construído.
- Testes
