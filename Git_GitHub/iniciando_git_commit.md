- Iniciar versionamento (git add)
- Criar um commit (git commit)

Passo a passo:

- Criar uma pasta para o projeto
- dentro da pasta criar outra com o nome do projeto
- entrar ne pastar e colocar o comando git init
- Após isso deve configurar o usuário dentro dessa pasta
  Essa parte somente quando não tem usuário cadastrado se tiver pode pular
- Comando: git config --global user.email "orafael49@gmail.com" (configurar o e-mail do autor)
- Comando: git config --global user.name rafael (Configurar nome do autor)

- Criar qualquer arquivo ou projeto
- git add * : para adicionar arquivos criados
- git commit -m "qualquer texto" : salvar as alterações
