# Etendendo como o GIT funciona por baixo dos panos

## SHA1
A sigla SHA significa Secure Hash Algorithm, é um conjunto de funções hash
criptográficas projetadas pela NSA (Agência de segurança Nacional dos EUA)

A encriptação gera um conjunto de characteres identificador de 40 dígitos.
É uma forma curta de representar um arquivo.
