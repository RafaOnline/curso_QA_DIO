# Comandos básicos

## Windows
- **dir** : Listar as pastas do local atual
- **cd** : Navegas entre as pastas
  - **cd ..** : volta uma pasta anterior
- **cls** : Limpar terminal
- Tecla **TAB** : Completa as palvras quando for digitar
- **mkdir** : Criar uma pasta
- **echo** : Printa alguma palavra no terminal
  - **echo hello > hello.txt** : vai criar um arquivo .txt escrito hello
- **del + nome da pasta** : Deleta todos os arquivos internamente na pasta
- **rmdir** : Remove totalmente o Diretório/Pasta
